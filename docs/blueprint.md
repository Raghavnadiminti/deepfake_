# **App Name**: DeepFake Detective

## Core Features:

- Image Upload: Allow users to upload images for analysis.
- Deepfake Detection: Use Reality Defender's API to analyze uploaded images and determine if they are deepfakes, using your API key.
- Detection Result Display: Clearly display the results of the deepfake detection analysis to the user.

## Style Guidelines:

- Primary color: Dark purple (#624CAB) to convey authority, seriousness, and security in image authentication.
- Background color: Light gray (#EEEEEE), creating a neutral backdrop for the analysis results to stand out.
- Accent color: Teal (#4ABCB5) for the upload button and highlighted elements, drawing attention to interactive features.
- Font: 'Inter' sans-serif for both headlines and body text to give a modern, neutral look. 
- Use simple, clear icons for upload, analysis, and results to enhance user understanding.
- Single-page layout with a prominent image upload area at the top, followed by the analysis results displayed below.
- Subtle loading animation while the image is being analyzed, providing feedback to the user that the process is running.
